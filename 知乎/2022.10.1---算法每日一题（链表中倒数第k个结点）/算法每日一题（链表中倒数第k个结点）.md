## **题目描述：**

**输入一个链表，输出该链表中倒数第k个结点。**

**示例1：**

**输入：1,{1,2,3,4,5}**

**返回值：{5}**

## **解题思路：**

**这是一道经典的快慢指针问题，设置一对快慢指针即可！**

**代码：**

```
struct ListNode* FindKthToTail(struct ListNode* pListHead, int k )
 {
    // write code here
    struct ListNode* fast = pListHead, *slow = pListHead;
    while (k--)
 {
        if (fast == NULL)
        {
            return NULL;
        }
        fast = fast->next;
    }
    while (fast) {
        fast = fast->next;
        slow = slow->next;
    }
    return slow;
}
```
