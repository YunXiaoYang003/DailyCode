### **以下代码均经过牛客测试，均正确，请放心测试**

###  

**代码：**

```
#include <stdio.h>

int main() {
    int arr[3]={0};

    int max=arr[0];
    for(int i=0;i<3;i++)
    {
        scanf("%d",&arr[i]);
        if(max<arr[i])
        max=arr[i];
    }
    
    printf("%d",max);   
    return 0;
}
```
